# 当前进度说明

**文档日期：** 2026-04-19  
**项目：** 爱的真心话（SincereWordsOfLove）

---

## 一、目标与范围

- 提供可与 AI 对话的网页，接入**通义千问**（DashScope 兼容 Chat Completions）。
- AI 能力：将「父母对孩子说的话」转写为**父母内心的真实想法**（品牌化称呼：**爱的真心话**）。
- 产品约束：**50 字以内（含标点）**、柔和、不啰嗦；实现上由服务端 `SYSTEM_PROMPT` 强制执行。

## 三、技术栈快照

- **运行时：** Node.js ≥ 18（ESM、`fetch`）
- **HTTP：** 原生 `node:http`，无 Express
- **WebSocket：** `ws`（语音桥接升级与百炼上游）
- **前端：** 原生 HTML / CSS / JS（无构建步骤）
- **大模型：** 阿里云 DashScope OpenAI 兼容接口 `/chat/completions`
- **语音识别：** 百炼实时语音识别 WebSocket（默认 `paraformer-realtime-v2`）

---

## 四、已知限制与注意点

1. **密钥**：未配置 `DASHSCOPE_API_KEY` 时，`/api/chat` 返回 500，页面会提示错误。
2. **区域**：API Key 与 `DASHSCOPE_BASE_URL` 需与阿里云账号区域一致，否则可能鉴权或路由失败。
3. **字数**：模型输出依赖提示词约束，极端情况下可能出现略超 50 字；若需硬截断可在服务端后处理（当前未做）。
4. **并发与安全**：当前为本地个人使用场景；若公网部署需另行考虑鉴权、限流与 HTTPS。
